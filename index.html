🏗 My Projects

💊 PhysiBot

PhysiBot is an assistive AI-based chatbot that provides a preliminary health prognosis based on a machine learning model built using the IBM Watson Studio. The user has to input basic health information and symptoms and the app returns a preliminary assessment and the directions to the nearest healthcare facility for a proper diagnosis. PhysiBot was built with the aim to make foreign healthcare systems less daunting to navigate by tourists and especially, international students.

🌽 AgGro

AgGro is a web application that aggregates and automates ordering groceries for individuals living in shared spaces. It allows users to create shared grocery lists, set reminders, track the expiry of items bought and split costs easily amongst members in the group. AgGro was built with the aim to reduce conflict amongst people living in a shared space, facilitate easier bill splitting, reduce redundancy while shopping and save time as users will make fewer trips to the grocery store.

📚 Lingz

Lingz is a service that translates Massive Online Open Courses(MOOCs) and other video tutorials into different languages so that they are available to a broader audience. Most online courses are only available in English, but only 20% of the World's population is fluent in it. Lingz makes these courses accessible to non-English speakers by providing translated audio and subtitles synced at the appropriate timestamps in the video tutorial.
